MDH Explosives Actions is a mod, created by Moerderhoschi for Arma 3. 

HOW THE ADDON WORKS?
You get the better holdAction menu entries to put down Explosives or you also can deactivate actions complete and put down and activate explosives manually.

IS THE ADDON MP COMPATIBLE?
Yes, in MP only the Client/Player needs the Addon.
